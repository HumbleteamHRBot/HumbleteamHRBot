# 🤖 Онбординг-бот для Telegram

Telegram-бот для помощи новым сотрудникам, работающий на базе Claude AI.

## Возможности

- 💬 Ответы на вопросы об онбординге через Claude AI
- 📚 База знаний с поиском по ключевым словам
- ❓ FAQ — частые вопросы новичков
- 🔄 История диалога для контекстных ответов
- 🛡️ Fallback на локальную базу при недоступности API

## Структура

```
├── bot.py              # Главный файл — Telegram-бот (aiogram)
├── ai_handler.py       # Интеграция с Claude API
├── knowledge_base.py   # База знаний, FAQ и поиск
├── requirements.txt    # Python-зависимости
├── Procfile            # Конфиг для Railway
└── README.md
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен от @BotFather |
| `ANTHROPIC_API_KEY` | API-ключ от console.anthropic.com |

## Деплой на Railway

1. Загрузите код в GitHub-репозиторий
2. Подключите репозиторий к Railway
3. Добавьте переменные окружения (Variables)
4. Railway автоматически задеплоит бота

## Команды бота

- `/start` — приветствие и главное меню
- `/faq` — частые вопросы
- `/kb` — база знаний по разделам
- `/reset` — очистить историю диалога
- `/help` — справка
